# Overview
Codes for ESP32's smart irigation system to automate the watering process.

## Specification
Intended for ESP32 Dev board with built-in Wi-Fi. The board support monitoring three pots at once. The board can also be remotely controlled (limited).

## Sensors
Currently, the board only controls three soil moisture sensors at once.

## Future Improvement
The addition of soil acidity, light, and, temperature sensor can be integrated into the system.
